## Description

**Atomic Design UI** aims to provide a modern and refined interface to compliment and adapt to any dark syntax-theme. In addition, this theme also includes a custom icon set, modular settings, and improved integration of some of the most popular [Atom packages](https://atom.io/packages).  

You can download and learn more about this syntax-theme on [GitHub](https://github.com/tterb/Atomic-Design-UI).
